THE DEPLOYMENT DANCE

Choosing Render and Vercel

Render's free tier offers PostgreSQL databases and Python runtime, making it ideal for FastAPI backends. Vercel's GitHub integration provides instant deployments for React apps.

The CORS Configuration Challenge

One major hurdle: Cross-Origin Resource Sharing (CORS). Initially, my frontend couldn't communicate with the backend because the API wasn't configured to accept requests from the Vercel domain.

The fix required adding environment variables to allow the frontend origin:
```python
allowed_origins = [
    "https://hrms-lite-eight-peach.vercel.app",
    "http://localhost:5173",
]
```

The Python 3.13 Compatibility Issue

Here's where things got interesting. Render uses Python 3.13, but psycopg2-binary (the PostgreSQL driver) wasn't compatible. After multiple failed attempts to build wheels, I made a pragmatic decision: switch to SQLite.

While SQLite on Render's ephemeral filesystem means data resets when the service sleeps (after 15 minutes of inactivity), it allowed the project to work immediately. For a demo/interview scenario, this tradeoff was acceptable.

LESSONS LEARNED

1. Start Simple, Then Optimize

My initial attempt at a complex layered architecture with separate service and repository layers caused import issues. I simplified to a flatter structure with models, schemas, and CRUD operations in one level. This reduced complexity without sacrificing maintainability.

2. Environment Configuration is Critical

Spending time upfront on proper environment variable management saved hours of debugging. Using python-dotenv for local development and Render's environment dashboard for production kept credentials secure.

3. Documentation Matters

Even under time pressure, I added docstrings to functions and comments explaining "why" not just "what." This helped when I needed to refactor quicklyâ€”my future self understood the reasoning behind decisions.

4. Test the Full Flow Early

I deployed the backend as soon as I had a basic endpoint working. This caught CORS issues and database connection problems early, rather than at the 11th hour.
